============================================

> my_package.py contains 

dictionary that maps spoken word to written word.
EX : {"tripple a" : "aaa"}

add(a,b) this function adds new values to dictionary.
EX : dict.add("two dollor", "$2") gives new_dict : {"tripple a" : "aaa", "two dollor", "$2" }
